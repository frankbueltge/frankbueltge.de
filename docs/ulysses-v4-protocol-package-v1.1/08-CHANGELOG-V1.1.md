# Changelog — package revision 1.1

This revision corrects the human-workload and auto-land model in the original package.

## Changed

- Replaced project-by-project human score approval with a versioned standing delegation.
- Clarified that Frank is not the daily project manager or routine branch reviewer.
- Allowed Ulysses to initiate, execute, revise, archive and kill ordinary projects within mandate.
- Replaced the old merge-plus-site-dispatch workflow with a safe, allowlisted research auto-land proposal.
- Kept curated publication human-only.
- Added explicit separation between public research repository records and curated Atelier works.
- Added mandate and operator-role documents, escalation conditions and no-response behaviour.
- Updated templates, validator, acceptance tests, decision-log patch and Claude handoff accordingly.

## Unchanged

- Generic nightly production remains superseded.
- Protocol v3 and historical records remain preserved.
- Rights-sensitive, irreversible, costly and constitutional decisions remain human-accountable.
- No shadow run, multi-agent council, novelty score or daily human review is introduced.
