# Ulysses / Atelier

Ulysses is a situated artistic research practice by Frank Bültge, developed through documented human–machine operations.

A standing human delegation defines budgets, permissions, protected boundaries and escalation conditions. Within it, models and tools may identify project candidates, initiate projects, research, compare, transform, build, revise, archive and terminate work without daily human approval.

The work begins from concrete materials, encounters, claims, instruments or situations in which something consequential does not fit. Completion or automatic repository integration does not create an obligation to publish.

The repository is an inspectable research archive. It may contain active projects, studies, failed or killed lines and publication candidates. The curated Atelier works surface remains separate and requires an explicit human-approved publication manifest.

## Operating principles

- Project, not generic session.
- Standing delegation, not daily approval.
- Concrete source situation, not a grand concept alone.
- Material or external resistance, not model randomness as a substitute.
- Safe auto-land for reversible research records.
- Curated publication remains human-only.
- Rights, provenance, affected publics and refusal are constitutive conditions.

The earlier nightly model and merge-to-site-dispatch coupling are superseded. Their records remain visible as part of the experiment’s history.
