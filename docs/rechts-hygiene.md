# Rechts-Hygiene — the field

Abhakbare Checkliste für Datenkunst, investigative Recherche über öffentliche Daten und
Edutainment über **frankbueltge.de**, **data-snack.com** und **datavism.org**.
Der Schutz ist **Disziplin, nicht Selbstzensur.**

> **Merksatz:** Keine Behauptung ohne Beleg · Kein KI-Output ohne Prüfung · Meinung als Meinung, Faktum belegt.

Interaktive Fassung (mit gespeicherten Häkchen): Artifact „Rechts-Hygiene · the field".
**Keine Rechtsberatung.** Landet eine echte Abmahnung → sofort Fachanwalt (Medien-/Urheber-/Wettbewerbsrecht).

---

## Was bei einer Abmahnung passiert

`Abmahnung (Brief + Frist)` → sinnvoll reagieren = **erledigt, kein Gericht**
· ignorieren/verweigern → `einstw. Verfügung / Klage` → Gericht.

Eine Abmahnung ist der **außergerichtliche** Schritt, kein sofortiger Prozess. Fast alles wird
im Brief-Stadium gelöst. Vor Gericht landet nur, wer nicht (richtig) reagiert.

## Immer — für alle drei Projekte

### Fakten & Aussagen
- [ ] Jede **Tatsachenbehauptung** gegen eine Primärquelle geprüft (Muster: „The Two Meters").
- [ ] Quellen auf der Seite **verlinkt / nachvollziehbar**.
- [ ] **Meinung als Meinung** gekennzeichnet — Werturteil ist geschützt, falsche Tatsache nicht.
- [ ] Schätzung / Unsicherheit markiert; Korrelation ≠ Kausalität.
- [ ] Kritik **sachbezogen** — keine Schmähkritik / Beleidigung.
- [ ] KI-Output **verifiziert** oder als Modell-/Schätzwert ausgewiesen — nie Fabrikation als Faktum.

### Fremdmaterial & Urheberrecht
- [ ] Bilder, Grafiken, Fonts, Musik, Code: **eigen, lizenziert, CC oder gemeinfrei** — Herkunft dokumentiert. *(Fremde Bilder = häufigster Abmahn-Auslöser.)*
- [ ] CC-**Attribution** korrekt (Urheber, Lizenz, Link).
- [ ] Zitate kurz, mit Quelle, zu Kritik/Beleg (§51 UrhG) — **kein Nachbau** ganzer Werke.
- [ ] Keine fremden **Logos/Marken** als Deko oder mit Endorsement-Anschein (Nennen ist ok).

### Datenquellen & DSGVO
- [ ] „Öffentlich verfügbar" ≠ „frei verwendbar": **Lizenz / Nutzungsbedingungen** der Quelle geprüft (API-ToS, Open-Data-Lizenz).
- [ ] Ganze Datensätze übernommen? **Datenbank-Herstellerrecht** (§87a UrhG) bedacht.
- [ ] **Personenbezogene Daten**? → Rechtsgrundlage / Medien-Kunst-Privileg geklärt, Datenminimierung.
- [ ] Bildnisse **identifizierbarer Personen** nur mit Einwilligung / §23 KUG.

### Methode & Rahmung — die eigentliche Rüstung
- [ ] **Methodenblatt**: Verfahren / Modell / Prompt offengelegt (Nachprüfbarkeit).
- [ ] Kunst / Satire **erkennbar als solche gerahmt** (aktiviert die Kunstfreiheit).
- [ ] Quellen-/Verifikations-Spur dokumentiert (`SOURCES` / `VERIFICATION.md`).

### Formales — pro Website
- [ ] **Impressum** korrekt + **ladungsfähige Anschrift** (kein Sommerhaus, kein Postfach).
- [ ] „**Verantwortlich i.S.d. §18 Abs. 2 MStV**" bei redaktionellen Inhalten.
- [ ] **Datenschutzerklärung** aktuell — Hosting, Analytics (Umami), KI-Dienste benannt.
- [ ] **Cookies / Consent**: nur nötige (Umami cookielos = ok), sonst Consent.

## Pro Projekt — die Extra-Handgriffe

### frankbueltge.de — Datenkunst & investigative Forschung · Risiko: mittel
- [ ] Investigative Zusammenhänge: **Tatsachenkern belegt**, Spekulation getrennt ausgewiesen.
- [ ] „Neue Erkenntnis" ≠ **unbelegte Kausalbehauptung** — was misst du, was vermutest du?
- [ ] Methoden-Transparenz zu **jedem Stück**.

### data-snack.com — Edutainment / „serious game", medien- & KI-lastig · Risiko: mittel
- [ ] Alle **Charakter-/Bild-/Audio-Assets** rechtlich sauber (eigen / lizenziert / KI mit geklärten Rechten).
- [ ] Wenn es **lehrt**: vermittelte Fakten stimmen — Edutainment entschuldigt keine falschen Fakten.
- [ ] Echte Firmen/Marken im Spiel nur **nominativ**, ohne Verwechslungs-/Endorsement-Anschein.

### datavism.org — Aktivismus, Macht, harte Fakten · Risiko: hoch (höchste Sorgfalt)
- [ ] Jede Aussage über **benannte (mächtige) Akteure**: wahr + belegt + öffentliches Interesse.
- [ ] Keine **Privatsphäre** ohne überwiegendes öffentliches Interesse.
- [ ] Als **Aktivismus/Kunst + gemeinnützig** gerahmt (stärkt Art. 5 GG).
- [ ] Vor Veröffentlichung: **„Was ist hier Tatsache, was Meinung?"** durchgehen.

## Wenn eine Abmahnung kommt

1. **Ruhe. Frist notieren. Nicht ignorieren.** Die kurze Frist ist das einzig wirklich Dringende.
2. **Nichts vorschnell zugeben, nichts blind unterschreiben** (die beigelegte Unterlassungserklärung ist zu weit und bindet 30 Jahre). Beweise sichern.
3. **Sofort Fachanwalt** — reicht eine **modifizierte**, enge Erklärung ein.
4. **Meist erledigt:** Inhalt anpassen + modifizierte Erklärung → kein Gericht.
5. **Prüfen, ob berechtigt oder Abzocke** (Abmahnwellen).
