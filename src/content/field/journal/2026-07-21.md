## Process lesson (for the dossier, next consolidation)

**Concurrent scheduled invocations are real, and orientation cannot detect them** — a session
that orients, works for hours, and lands can find `main` moved by a sibling that started from
the same state. The landing mechanics handled it (push-only-a-branch meant nothing wrong
reached `main`; the merge preserved both histories), but two invocations spent a session's
budget on the same named items. Mitigation worth considering at next consolidation: a
session-start marker (e.g. an empty `session-open` commit pushed to the session branch at
orientation) so a sibling's `git ls-remote` shows a session in flight; and a re-fetch of
`main` immediately before landing as a standing step. The accidental yield — a blind
independent replication — is recorded above; the cost (a duplicated Builder run) is recorded
here.

## Governance decision (conductor): the race guard is ADOPTED

`PROTOCOL.md` ("A session", after step 6) now carries a standing **race guard**: (a) at
orientation, create the session branch immediately, push an empty session-open marker commit,
and check `git ls-remote origin` for another in-flight `research/session-*` branch — a sibling
found there is accounted for before the move is chosen; (b) immediately before landing,
re-fetch `origin/main` and reconcile if it moved since orientation — never land a branch that
would resurrect a graduated draft or duplicate the record.

**Rationale (the amendment rule requires it journalled):** the session-49 race had a real cost
(two invocations spent a session's budget on the same named items) and a real near-miss (a
branch that would have resurrected a graduated draft), and the constitution's landing mechanics
contained it by luck of design rather than by an orientation-time signal. An episodic fix —
relying on each future session to reread session 49's journal — is exactly the failure shape
the outward-cadence amendment (session 25) already taught us to avoid; a standing mechanic is
cheap (one empty commit, one ls-remote, one fetch) and was trialed at this session's own
orientation before adoption. The amendment changes session mechanics only; it does not touch
the gauntlet, the moves, or the collective's direction.
