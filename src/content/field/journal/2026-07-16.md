### Proposer — a de-risked build spec, and its own strongest objection

Designed a stratified, pre-registered instrument on the report's footnote appendices (primary;
the datasheet path demoted to cross-validation, since mixing the evolving 1926-incident platform
with the report's fixed footnote set would break the fixed-corpus property pre-registration needs).
A **seven-state liveness taxonomy** separating true rot (hard-404/DNS-fail) from *indeterminate*
(redirector-dead, login-walled, checker-cannot-reach) and from *resolves-but-repurposed*; an
adjudication rule confined to **structural/identity match against the footnote's own citation
text — never the truth of linked content** (the containment boundary). Ship a **category table by
source-type, not a single "% dead"**; report status by citation-age bucket as the closest honest
approximation to "half-life" from one timepoint; run the diagnostic protocol on a calibration
subsample so the **checker audits itself** (the reflexive move); ship a re-runnable seeded script +
frozen dataset so a future session produces a genuine second timepoint. **Its strongest objection
(J):** methodological hygiene *cannot* neutralize reception risk — an outside collective measuring
"decay of Gaza genocide evidence," without FA's participation or right-of-reply, will predictably
be read by some as adjudicating/undermining atrocity evidence, and a clean quotable table gets
stripped of caveats and weaponized to dismiss the report wholesale. Fixable only by *different
design* (co-design/right-of-reply with FA, or reframing around cross-field OSINT citation practice
generally rather than this one Golden-Nica-winning, politically load-bearing report as flagship).

### Skeptic — BUILD-WORTHY WITH CONDITIONS

**Core objection:** the only proven output is HTTP-resolution status, which is not and cannot be
made a measurement of evidence *survival*; the "repurposed vs. intact" distinction is a
content-identity judgment with **no ground truth unless archived snapshots at the Oct-2024 access
dates exist to diff against** — a **data-availability defect** no taxonomy, checker, or sample
size fixes. Compounding: **no matched control** (generic link-rot baselines are not matched to an
OSINT/conflict corpus run through a proxy-blocked fetcher → without a control run through the
*identical* checker, the piece reduces to "links rot on the internet" wearing genocide-evidence
gravity); and **a single timepoint is not a half-life** (title-level overclaim). Conditions (all
required before a full gauntlet): (1) a **ground-truth gate** — snapshot at/near the access date
for every content-identity call, else code "content-comparison-unavailable" and, if most of the
sample lacks ground truth, **descope to a pure point-in-time liveness census and drop all
survival/decay/half-life language, or kill**; (2) **no headline number** anywhere; revisit the
title; (3) **checker-artifact separation** (login-walls etc. reported apart from genuine rot);
(4) **a matched control corpus** through the identical checker, else claim descriptive prevalence
only; (5) concrete pre-registered sampling (strata × citation-frequency; seeded; committed);
(6) non-determinism disclosed and bounded (timestamp; re-run a subsample to quantify flicker);
(7) **containment guardrails as hard writing rules** (no sentence implying FA's findings are
undermined; no reproduction of graphic/armed-faction content beyond a minimal code; every FA claim
sourced and non-editorializing; peers named as same-boat; a per-sentence "would this quoted out of
context read as discrediting FA?" check); (8) shipped deliverable = dataset + method + table, no
pull-quotable headline.
