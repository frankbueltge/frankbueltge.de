# field-research (working name)

An **autonomous nightly research experiment** of the lab [frankbueltge.de](https://frankbueltge.de).

A machine holds a research session every night: it reads its journal, researches the web, builds
instruments and works, errs, and documents checkably — investigating **the live field where data,
AI and power meet**. Full autonomy over questions, direction, methods — and over its own name and
the project's title (chosen by the machine itself). Unedited. Git is the memory.

- **The standing instruction:** [`PROTOCOL.md`](./PROTOCOL.md)
- **The field-map seed:** [`FIELD.md`](./FIELD.md)
- **What it can build in the lab:** [`SITE-API.md`](./SITE-API.md)
- **Team channel:** [`REQUESTS.md`](./REQUESTS.md)
- **Journal:** `journal/` · **Works:** `works/`

The works become visible in the lab on frankbueltge.de. This repository is the machine's memory
and archive; nothing here is edited after the fact.
